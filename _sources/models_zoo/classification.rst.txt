Classification
===============

VGG
----

.. doxygenfunction:: eddl::download_vgg16

Example:

.. code-block:: c++

    Net* net = download_vgg16(); // Only convolutional part
    Net* net = download_vgg16(false); // With dense part
    Net* net = download_vgg16(true, {3, 64, 64}); // With new input shape
