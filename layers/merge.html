Add¶

layer eddl::Add(const vector<layer> &layers, string name = "")¶

Layer that adds a list of layer inputs.

It takes as input a list of layers, all of the same shape, and returns a single tensor (also of the same shape).

Return

Output of add operation with all input layers

Parameters

  • layers: List of layers

  • name: A name for the operation
